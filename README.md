Built a Python-based Phishing Email Detector 
as part of my cybersecurity learning journey.

What it does:
- Analyzes email text for suspicious patterns
- Detects urgency tactics, fake requests, threats
- Calculates Risk Score from 0 to 100
- Returns HIGH / MEDIUM / LOW risk level

Why it matters:
90% of cyberattacks start with phishing emails.
This tool helps detect them before it's too late!

Tech used: Python 3, String Analysis, 
Dictionary, Loops, Functions
